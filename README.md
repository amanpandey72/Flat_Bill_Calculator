# Flat_Bill_Calculator
# Flat Expense Split Calculator (Python)

This is a simple Python program that calculates how much **each person needs to pay** when sharing flat expenses such as rent, food, and electricity.

The program takes input from the user and splits the total cost equally among all people living in the flat.

---

## Inputs Required
The program asks the user to enter the following details:

- Total flat rent  
- Total amount spent on food/snacks  
- Total electricity units consumed  
- Charge per electricity unit  
- Number of persons living in the flat
